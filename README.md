# Xaloc Tanks!
Candidate technical test 

The project uses _Unity 2022.3.2f1_.

## Game controls

| Controls   | Player 1  | Player 2    |
|------------|-----------|-------------|
| Turn Left  | A         | Left Arrow  |
| Turn Right | D         | Right Arrow |
| Move Up    | W         | Up Arrow    |
| Move Down  | S         | Down Arrow  |
| Fire       | Space     | Enter       |
